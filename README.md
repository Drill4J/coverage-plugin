# Test2Code Plugin

Minimize your regression suite via Test Impact Analytics.
For more information see https://drill4j.github.io/

## How to Run

With Gradle:
`./gradlew run`

## How to Debug

change DRILL_AGENTS_SOCKET_TIMEOUT for your needs.

### Admin part

by remote debug with port 5006
